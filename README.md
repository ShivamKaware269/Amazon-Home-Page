# Amazon Landing Page Clone

A responsive front-end clone of the Amazon landing page built using **HTML5** and **CSS3**. This project focuses on recreating the modern Amazon homepage with a clean layout, responsive design, and attention to detail.

## Features

* Responsive homepage layout
* Amazon-inspired navigation bar
* Hero banner section
* Product category cards
* Footer with multiple sections
* Clean and organized HTML & CSS code
* Beginner-friendly project structure

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to strengthen front-end development skills by practicing semantic HTML, CSS layouts, Flexbox, responsive design, and UI replication.
